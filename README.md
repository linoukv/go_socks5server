# SOCKS5 代理服务器 - 项目总结

## 项目概述

这是一个功能完善的 SOCKS5 代理服务器，使用 Go 语言开发。该项目提供了完整的代理功能、用户管理系统、流量统计和 Web 管理界面，适用于生产环境部署。

## 技术栈

- **开发语言**: Go 1.25.0
- **数据库**: SQLite3
- **主要依赖**:
  - `github.com/mattn/go-sqlite3` - SQLite 数据库驱动
  - `golang.org/x/crypto` - 密码加密库（bcrypt）

## 核心功能

### 1. SOCKS5 代理服务

- **完整的 SOCKS5 协议实现**: 支持 CONNECT 和 UDP ASSOCIATE 命令
- **TCP 代理**: 高性能的 TCP 数据转发，支持流量统计
- **UDP 代理**: 低延迟的 UDP 数据包转发
- **连接优化**: 支持 TCP NoDelay、Keepalive 等参数优化

### 2. 用户认证系统

- **多种认证方式**: 支持无认证和密码认证两种模式
- **密码安全**: 使用 bcrypt 算法加密存储密码
- **用户管理**: 完整的 CRUD 操作（添加、删除、修改、查询）
- **用户限制**:
  - 最大并发连接数限制
  - 单 IP 最大连接数限制
  - 账户启用/禁用控制
- **分片架构**: 使用 16 个分片来减少锁竞争，提升高并发性能

### 3. 流量配额管理

- **多种配额周期**:
  - 每日配额
  - 每周配额
  - 每月配额
  - 自定义时间段配额
  - 无限制模式
- **实时配额检查**: 在数据传输过程中动态检查配额
- **自动配额重置**: 定期自动重置过期的配额

### 4. 数据持久化

- **SQLite 数据库**: 轻量级嵌入式数据库
- **异步写入队列**: 批量处理数据库写操作，减少锁竞争
- **数据完整性**:
  - 用户信息持久化
  - 流量统计持久化
  - 系统配置持久化
  - 管理员账户持久化

### 5. Web 管理界面

- **管理员登录**: 独立的管理员认证系统
- **用户管理面板**:
  - 添加/编辑/删除用户
  - 查看用户状态
  - 设置用户连接限制
  - 设置用户流量配额
- **服务器配置**:
  - 监听地址配置
  - 最大工作协程数配置
  - 单 IP 连接数限制配置
  - TCP Keepalive 周期配置
- **认证方式切换**: 在无认证和密码认证之间切换
- **实时统计**: 显示总用户数、活跃用户、总上传/下载流量

### 6. 性能优化

- **缓冲区池**: 复用内存缓冲区，减少 GC 压力
- **工作协程池**: 限制并发处理的连接数量
- **批量提交**: 流量统计和数据库写入采用批量策略
- **原子操作**: 高并发场景下使用 atomic 包保证数据一致性
- **内存管理**: 定期释放未使用的内存给操作系统

## 核心模块说明

### main.go

程序入口文件，负责：
- 初始化数据库
- 加载配置
- 启动 SOCKS5 服务
- 启动 Web 管理界面
- 处理优雅关闭信号

### server.go

服务器核心模块，实现：
- 监听客户端连接
- SOCKS5 协议握手
- 认证处理
- 连接管理
- 统计信息收集
- 用户数据持久化

### auth.go

认证和用户管理模块，提供：
- 用户 CRUD 操作
- 密码哈希和验证
- 连接数限制
- IP 连接数限制
- 流量配额管理

### database.go

数据库管理模块，封装：
- SQLite 连接管理
- 表结构初始化
- 用户数据持久化
- 系统配置存储
- 异步写入队列

### proxy.go

数据转发模块，实现：
- TCP 代理（带流量统计）
- UDP 代理（低延迟）
- 缓冲区池管理
- 用户配额检查

## 默认配置

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| 监听地址 | 0.0.0.0:1080 | SOCKS5 服务监听地址 |
| Web 管理地址 | 0.0.0.0:8080 | Web 管理界面监听地址 |
| 最大工作协程数 | 1000 | 限制并发连接数 |
| 单 IP 最大连接数 | 65535 | 单个 IP 的最大连接数 |
| 握手超时 | 10 秒 | SOCKS5 握手超时时间 |
| TCP Keepalive 周期 | 15 秒 | TCP 心跳检测周期 |
| 缓冲区大小 | 32KB | 数据传输缓冲区大小 |
| 数据库路径 | socks5.db | SQLite 数据库文件 |

## 安全特性

1. **密码安全**: 使用 bcrypt 算法加密存储密码
2. **时序攻击防护**: 对不存在的用户执行恒定时间的密码比较
3. **输入验证**: 严格验证用户名和密码格式
4. **连接限制**: 防止单个用户或 IP 占用过多资源
5. **流量配额**: 防止用户滥用带宽资源
6. **独立管理员系统**: Web 管理使用独立的管理员账户

## 使用场景

- **企业内网代理**: 为企业内部网络提供安全的代理服务
- **个人代理服务**: 个人使用的安全代理服务
- **流量管理**: 需要控制用户流量使用的场景
- **多用户环境**: 支持多用户并发访问的代理服务

## 扩展性

项目采用模块化设计，具有良好的扩展性：
- 易于添加新的认证方式
- 支持扩展新的数据库后端
- Web 界面易于定制和扩展
- 支持添加新的协议支持

## 总结

这是一个生产级别的 SOCKS5 代理服务器项目，具有完善的功能、优秀的性能和良好的可维护性。项目采用 Go 语言开发，充分利用了 Go 的并发特性，同时通过多种优化策略确保在高并发场景下的稳定运行。
