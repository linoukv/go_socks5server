# SOCKS5 代理服务器项目技术文档

## 1. 项目概述

这是一个基于 Go 语言开发的高性能 SOCKS5 代理服务器，提供完整的代理服务功能和直观的 Web 管理界面。该项目实现了标准 SOCKS5 协议，支持用户认证、流量统计、配额管理等企业级特性，适合个人和企业使用。

## 2. 核心功能

### 2.1 SOCKS5 代理服务
- 支持标准 SOCKS5 协议，包括 CONNECT 和 UDP ASSOCIATE 命令
- 高性能并发处理，使用工作池和缓冲区池优化内存使用
- 支持 TCP Keepalive 和 TCP NoDelay 等网络优化参数
- 实现单 IP 连接数限制，防止资源滥用

### 2.2 用户认证与管理
- 支持用户名/密码认证机制
- 用户连接数和 IP 连接数限制
- 基于 bcrypt 的密码安全存储
- 用户数据持久化到 SQLite 数据库

### 2.3 流量管理与配额
- 实时流量统计（上传/下载）
- 支持多种配额周期（每日/每周/每月/自定义）
- 配额自动重置机制
- 详细的流量使用报告

### 2.4 Web 管理界面
- 直观的用户管理界面（创建/编辑/删除用户）
- 实时仪表盘，展示服务器状态和流量统计
- 配额管理页面，设置和监控用户流量配额
- 系统配置管理，调整服务器参数

### 2.5 安全特性
- CSRF 防护
- 验证码系统，防止暴力破解
- 基于 IP 的速率限制，防止 API 滥用
- 会话管理和超时控制
- 安全的密码存储（bcrypt 加密）

## 3. 技术架构

### 3.1 主要模块

| 模块 | 职责 | 文件 |
|------|------|------|
| 核心服务 | SOCKS5 协议实现和连接管理 | server.go |
| 代理功能 | TCP/UDP 代理转发 | proxy.go |
| 认证系统 | 用户认证和权限管理 | auth.go |
| 数据库 | 数据持久化和配置管理 | database.go |
| Web 界面 | 管理后台和 API 服务 | web_server.go |
| 统计监控 | 流量统计和服务器状态 | stats.go |
| 工作池 | 并发控制和资源管理 | pool.go |
| 协议处理 | SOCKS5 协议解析和响应 | protocol.go |

### 3.2 技术栈

- **开发语言**：Go
- **数据库**：SQLite
- **Web 框架**：标准库 net/http
- **安全库**：golang.org/x/crypto/bcrypt
- **速率限制**：golang.org/x/time/rate
- **前端**：HTML5 + JavaScript

### 3.3 系统架构

1. **多层设计**：
   - 网络层：处理 TCP/UDP 连接
   - 协议层：解析和处理 SOCKS5 协议
   - 服务层：业务逻辑和用户管理
   - 存储层：数据持久化
   - 管理层：Web 管理界面

2. **并发模型**：
   - 使用 goroutine 处理每个客户端连接
   - 工作池限制并发数量，防止资源耗尽
   - 原子操作和互斥锁确保线程安全

3. **内存优化**：
   - 缓冲区池复用内存，减少 GC 压力
   - 快照机制减少锁持有时间
   - 定期内存释放，优化系统资源使用

## 4. 核心流程

### 4.1 服务器启动流程
1. 初始化数据库连接
2. 加载服务器配置
3. 创建 SOCKS5 服务器实例
4. 初始化用户认证系统
5. 启动 Web 管理界面（后台协程）
6. 启动 SOCKS5 代理服务（后台协程）
7. 等待终止信号并优雅关闭

### 4.2 客户端连接处理流程
1. 接受客户端连接
2. 检查 IP 连接数限制
3. 提交连接处理任务到工作池
4. 处理认证握手（可选）
5. 处理 SOCKS5 命令请求
6. 根据命令类型执行相应操作（CONNECT/UDP ASSOCIATE）
7. 建立代理连接并开始数据转发
8. 记录流量统计信息
9. 连接结束后清理资源

### 4.3 数据持久化流程
1. 每 10 秒自动保存用户数据到数据库
2. 使用快照机制减少锁持有时间
3. 原子操作读取流量统计数据
4. 确保数据一致性和可靠性

## 5. 部署与配置

### 5.1 环境变量

- `SOCKS5_DB_PATH`：数据库文件路径（默认：socks5.db）
- `SOCKS5_WEB_ADDR`：Web 管理界面地址（默认：:8080）

### 5.2 默认配置

- SOCKS5 服务端口：1080
- Web 管理界面端口：8080
- 默认管理员账号：admin / password123（首次启动时创建）

### 5.3 配置参数

| 参数 | 描述 | 默认值 |
|------|------|--------|
| ListenAddr | SOCKS5 服务监听地址 | 0.0.0.0:1080 |
| MaxWorkers | 最大工作协程数 | 1000 |
| MaxConnPerIP | 单 IP 最大连接数 | 65535 |
| HandshakeTimeout | 握手超时时间 | 10秒 |
| TCPKeepAlivePeriod | TCP Keepalive 周期 | 15秒 |
| TCPNoDelay | 是否禁用 Nagle 算法 | true |
| RecvBufferPool | 接收缓冲区池大小 | 32KB |
| SendBufferPool | 发送缓冲区池大小 | 32KB |
